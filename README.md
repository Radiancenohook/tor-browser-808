## Troubleshooting
| Problem | Solution |
|--------------------------------|-------------------------------------------------------|
| Browser doesn't start | Run as Administrator and disable antivirus temporarily |
| Tor connection not working | Enable privacy optimization in settings |
| Onion dark web issues | Update to latest version 2026 |
| Privacy / security / vpn missing | Reinstall Tor Browser and check compatibility |
| Windows 11 compatibility error | Update Windows and run the setup guide again |
| Browser crashes | Verify installation and clear Tor cache |
